Civic Hacking at Fletcher Free Library

{{ page.description }}

Everyone is welcome, and all skills are relevant. In particular, if you have any of these skills (beginner through advanced), we need you!

  • PHP, Ruby, or any other programming language
  • WordPress
  • Graphic design, web design, UI/UX, JavaScript, CSS
  • Copywriting, information architecture, project management

You can choose to join an existing Code for BTV team, or start a new project. We have a list of potential projects for you!

Hack the Stacks is organized by Code for BTV, a Code for America Brigade.
